## FastAPI 

1. `Pydantic ` 
   - It Automatically Verify the Body Data which came from the `POST/PUT/PATCH`
   - Ex:

```python 
     from pydantic import BaseModel
     from fastapi import FastAPI
     
     class bodyData(BaseModel):
        name : str 
        age : int 
     
     app = FastAPI()
     
     @app.post('/getData')
     def bodyFN(bodyData: bodyData):
        return {
            "name" : bodyData.name,
            "age" : bodyData.age     
        }
```
